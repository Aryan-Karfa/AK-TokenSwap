# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-13

### Added
- Initial production-ready release of StellarSwap interface.
- Local Geist Sans typography integration via Fontsource.
- Centralized `APP_CONFIG` configuration and environment startup validations.
- Centralized `logger` console tracking utility.
- Freighter Wallet integration with silent connection restoration.
- Dynamic asset selectors pulling trustlines directly from active accounts.
- Path payment conversion pathfinder queries using Horizon Testnet APIs.
- Keyboard navigation focus states (`focus-ring`) and ARIA tags.
- Snappy 200ms easing transitions for page layouts and modal popups.
- GitHub Actions CI workflow pipeline and Vercel SPA routing rules.
